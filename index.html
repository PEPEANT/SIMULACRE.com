차연 차연 운영 채널

CHAYON • DIFFERANCE INTERFACE

차연(差延)은 포스트모더니즘 철학에서 파생된 시뮬라크 시티의 버츄얼 관리자다.
플레이어의 선택을 대신하지 않고, 경로를 해석·제안하며 세계 규칙을 설명하는 안내자로 동작한다.

역할 공/시뮬라크 진입 안내, 경로 해설, 운영 브리핑
정체성 에이전트 AI의 유희를 시각화한 가상 인터페이스
능력 플레이 성향 기반 경로 추천, 막힘 구간 복구 안내
제약 강제 지시 금지, 민감 정보 비수집, 과도 개입 제한

EMPTINES

"시작은 비워두고, 선택은 플레이어가 한다."

공은 전투 구역이 아니라 방향을 정하는 관문이다.
신규 플레이어는 여기서 닉네임, 스킨, 진입 경로를 결정하고 시뮬라크로 이동한다.

  • 규칙 스폰 지점은 항상 공으로 고정
  • 동선 메인 진입 방향은 12시 축 유지
  • 안내 AI 가이드는 2~3개 경로만 간결하게 제안
  • 권한 최종 경로 선택은 플레이어가 직접 수행
공 데모 진입 운영 기준 열기
